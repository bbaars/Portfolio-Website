Projects

FilmFo

FilmFo is an iPhone movie database app utilizing The Movie Database. It allows the user to view Top Rated Movies, Actors, and TVShows. It has "cross reference" searching allowing the user to search for two or more actors to see a list of movies/tvshows they're in together. It also lets them search for something, and then search for the actor / characters within that media. (Let's say you're watching Game of Thrones and want to know who plays Jon Snow. You just search Game Of Thrones and Jon Snow, and it'll return Kit Harrington.) The app also lets you add a Movie or TV Show to your favorites / watchlist.


There also is a desktop version, that is written in Java and has the same functions as the iPhone version.
